Is EPFL Down
============

[![Build Status][travis-image]][travis-url]
[![NPM version][npm-image]][npm-url]

Check if EPFL is down

Install
-------

```bash
$ npm install -g is-epfl-down
```

Command Line
------------

```bash
$ is-epfl-down
```
